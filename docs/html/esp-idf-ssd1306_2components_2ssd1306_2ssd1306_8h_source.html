9 /* Control byte for i2c
10 Co : bit 8 : Continuation Bit
11  * 1 = no-continuation (only one byte to follow)
12  * 0 = the controller should expect a stream of bytes.
13 D/C# : bit 7 : Data/Command Select bit
14  * 1 = the next byte or byte stream will be Data.
15  * 0 = a Command byte or byte stream will be coming up next.
16  Bits 6-0 will be all zeros.
17 Usage:
18 0x80 : Single Command byte
19 0x00 : Command Stream
20 0xC0 : Single Data byte
21 0x40 : Data Stream
22 */
