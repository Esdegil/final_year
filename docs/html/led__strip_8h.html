◆ led_strip_get_pixel_color()

bool led_strip_get_pixel_color ( struct led_strip_t *  led_strip,
uint32_t  pixel_num,
struct led_color_t *  color 
)

Get the pixel color at pixel_num for the led strip that is currently being shown! NOTE: If you call set_pixel_color then get_pixel_color for the same pixel_num, you will not get back the same pixel value. This gets you the color of the pixel currently being shown, not the one being updated

If there is an invalid argument, color will point to NULL and this function will return false.
